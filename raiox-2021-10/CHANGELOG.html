Dados abertos do Raio-X da Administração Pública Federal

Histórico de versões

Versão 1.3.0 (julho/2021)

Tradução para inglês

Os metadados no Data Package agora estão disponíveis, além do original em português, também em idioma inglês. Isso inclui os títulos e descrição do conjunto de dados, de todas as tabelas (recursos) e de todos os campos de cada tabela.

Novos status para transformação digital

Foram incluídos no esquema da tabela de transformação digital novos valores possíveis para os campos situacao, situacao_raiox e integracao_pagtesuro.

Mudanças de nomes na tabela de cargos alocados

A tabela cargos-funcoes-alocados-ocupados.csv agora se chama cargos-funcoes-distribuidos-ocupados.csv. Além disso, os nomes dos campos foram alterados para ficarem harmonizados com a nomenclatura usada no painel do Raio-X.

Versão 1.2.0 (junho/2021)

Publicação retroativa a partir de janeiro/2020

Foram incluídos arquivos zip contendo as extrações dos dados do Raio-X a partir do mês de janeiro de 2020. Algumas tabelas, entretanto, ainda não estão presentes nesses arquivos, considerando que os dados para esses períodos não estão disponíveis nos bancos de dados que dão origem ao painel do Raio-X da APF. Eventualmente, essas lacunas poderão ser preenchidas no futuro, caso seja viável encontrar fontes de dados retroativos que alimentem o banco de dados. Os dados que não contemplam o período completo são os seguintes:

  • transformação digital — disponível a partir de fevereiro/2020
  • pessoal: força de trabalho; pessoal: cargos e funções (perfil e alocação/ocupação); pessoal: alocação de servidores — disponível a partir de fevereiro/2020
  • soluções de modernização — disponível a partir de fevereiro/2020
  • patrimônio da união — disponível a partir de abril/2020

Os demais dados contemplam o período completo a partir de janeiro/2020.

Correções na tabela de pessoal força de trabalho

A tabela de pessoal – força de trabalho estava sendo extraída com referência incorreta de data. O problema foi corrigido e os dados retroativos extraídos.

Além disso, as linhas referentes ao Banco Central retornavam valores inválidos para o campo sexo. Como o órgão não fornece essa informação, esses casos foram preenchidos com o valor NÃO INFORMADO.

Correção nas quantidades de cargos e funções alocados e ocupados

As quantidades de cargos alocados não estavam sendo contabilizadas corretamente, o que também afetava o número de cargos disponíveis. O problema foi corrigido.

Correção nos dados de perfis de ocupantes de cargos e funções

Constavam agrupamentos de servidores que não ocupam, de fato, cargos e funções. Esses agrupamentos foram retirados, pois estão fora do escopo definido para esta tabela. Além disso, eram redundantes, já que há outra tabela para o perfil da força de trabalho.

Correções nos dados de transformação digital

Foram corrigidos alguns erros no processamento das tabelas de transformação digital: * alguns valores numéricos estavam ausentes (campo vazio) nas tabelas e agora estão devidamente preenchidos; * na previsão de semestre de entrega da digitalização do serviço, não constavam corretamente os serviços com entrega prevista a partir de 2021. Agora esses valores foram corrigidos. Além disso, essa coluna agora pode ser preenchida apenas nas situações Pactuado, Preparação, Automação, Implantação e Transformado.

Versão 1.1.0 (abril/2021)

Alteração de metodologia de cálculo do custeio

Em fevereiro de 2021, a Secretaria de Orçamento Federal mudou a metodologia de cálculo do custeio administrativo, conforme descrito na nota explicativa disponível no Boletim do Custeio Administrativo.

Com isso, foram atualizados os cálculos do custeio, inclusive retroativamente, referentes ao período de 2011 a 2020. Os dados do Raio-X da Administração Pública Federal foram atualizados para refletir essa mudança.

Assim, serão substituídos os arquivos zip dos meses a partir de agosto/2020, alterando as tabelas de custeio, com os novos dados, de maneira a possibilitar a comparação de dados em toda a série histórica.

Campos adicionais no patrimônio

Na tabela patrimonio-uniao.csv, foram acrescentados os campos municipio_nome, municipio_cod_ibge e uf conforme descritos no modelo de dados e descritor Data Package.

Correção na tabela de pessoal, cargos e funções

A tabela cargos-funcoes-perfil.csv possuía algumas agregações incorretas que geravam valores inválidos no campo sexo.

Além disso, em 69 linhas da tabela, vários campos figuravam com valores nulos.

Foi criada uma nova extração para a tabela que corrige esses problemas.

Correção na tabela de soluções de modernização

Alguns órgãos figuravam de forma duplicada nessa tabela. Uma nova lógica de extração dos dados corrige esse problema.

Versão 1.0.0 (agosto/2020)

Versão inicial, contendo:

  • Arquivo zip do mês, com os seguintes arquivos:
  • datapackage.json – arquivo com os metadados e descrição das tabelas no formato da especificação Data Package do Frictionless Data, incluindo os tipos de dados das colunas de cada tabela
  • transformacao-digital.csv – tabela com os dados sobre transformação digital
  • pessoal-forca-trabalho.csv – tabela com os dados sobre a força de trabalho
  • solucoes-modernizacao.csv – tabela com os dados sobre as soluções de modernização
  • patrimonio-uniao.csv – tabela com os dados sobre o patrimônio (imóveis) da União
  • custeio-administrativo.csv – tabela com os dados sobre o custeio administrativo
  • cargos-funcoes-perfil.csv – tabela com os dados sobre os perfis de ocupantes de cargos e funções
  • cargos-funcoes-alocados-ocupados.csv – tabela com os dados sobre os cargos e funções que estão alocados e ocupados
  • orcamento.csv – tabela com os dados sobre o orçamento federal
  • alocacao-servidores.csv – tabela com os dados sobre a alocação de servidores públicos
  • modelo de dados no formato PDF, descrevendo as tabelas

Os arquivos zipados também serão disponibilizados individualmente, fora do arquivo zip. Estes serão substituídos a cada atualização, mantendo sempre os dados do mês mais recente.

A cada mês será gerado um novo arquivo zip, mantendo os arquivos zip dos meses anteriores. Assim, para comparar diferentes meses, será necessário fazer o download dos arquivos zip dos meses anteriores.
